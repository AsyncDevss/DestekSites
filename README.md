# SynthetixWeb - Cyberpunk Discord Bot Landing Page

SynthetixWeb, Discord botlarınızı veya topluluk projelerinizi modern bir siber estetikle tanıtmanız için tasarlanmış, tamamen dinamik ve tek sayfalık bir web arayüzüdür. JavaScript nesnesi üzerinden kolayca özelleştirilebilir yapısı sayesinde HTML koduna dokunmadan tüm içeriği değiştirebilirsiniz.

## Özellikler

*   **Cyberpunk & Neon Tema:** Neon mavi ve pembe tonlarıyla zenginleştirilmiş koyu mod arayüzü.
*   **Dinamik Yapı:** Bot ismi, istatistikler, özellikler ve S.S.S. alanları tek bir JavaScript nesnesinden beslenir.
*   **Etkileşimli Arka Plan:** Canvas tabanlı, performansı yormayan hareketli ağ (particle) sistemi.
*   **Tam Mobil Uyum:** Responsive tasarım sayesinde telefon, tablet ve masaüstü cihazlarda kusursuz görünüm.
*   **Modern Akordeon S.S.S:** Sıkça sorulan sorular için animasyonlu açılır-kapanır menü sistemi.
